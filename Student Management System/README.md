# 🎓 Student Management System

A simple command-line **Student Management System** built using **Python**. This project allows users to manage student records with CRUD operations and stores data permanently using a JSON file.

## ✨ Features

- ➕ Add Student
- 👀 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student
- 🏆 Show Topper
- 📊 Calculate Average Marks
- 💾 Automatic Data Saving using JSON

---

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/SanjayLadumor/Python-Projects.git
```

2. Navigate to the project folder

```bash
cd "Student Management System"
```

3. Run the program

```bash
python main.py
```
