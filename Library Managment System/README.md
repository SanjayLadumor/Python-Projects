# 📚 Library Management System (Python)

A simple **console-based Library Management System** built using **Python**. This project demonstrates the use of Python fundamentals such as functions, lists, dictionaries, loops, conditional statements, and user input by implementing a real-world CRUD (Create, Read, Update, Delete) application.

---

## 🚀 Features

- ➕ Add a new book
- 📖 Display all books
- 🔍 Search a book by ID
- 📥 Borrow a book
- 📤 Return a book
- ❌ Delete a book
- 🆔 Prevent duplicate Book IDs
- 📋 Interactive menu-driven interface

---

## 🛠️ Technologies Used
