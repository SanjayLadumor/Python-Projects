# 🏦 Bank Management System (Python)

A simple **console-based Bank Management System** built using **Python**. This project simulates basic banking operations and demonstrates the use of Python fundamentals such as functions, dictionaries, loops, conditional statements, and user input.

---

## 🚀 Features

- 🆕 Create a new bank account
- 💰 Deposit money
- 💸 Withdraw money
- 👤 View account details
- 🔍 Search account by Account Number
- ❌ Delete an account
- 📋 Interactive menu-driven interface

---

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Lists
- Functions
- Loops
- Conditional Statements
