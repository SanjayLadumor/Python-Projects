# ☕ Coffee Vending Machine (Python)

A simple **console-based Coffee Vending Machine** built using **Python**. The program simulates the working of a coffee vending machine where users can order drinks, check resources, and make payments.

This project helped strengthen problem-solving skills and procedural programming concepts.

---

## 🚀 Features

- ☕ Order Coffee
- 💵 Accept coin payments
- 🧮 Calculate total payment
- 💰 Return change automatically
- 📦 Check ingredient availability
- 📊 Display available resources
- 💲 Track total profit
- 🚪 Exit the vending machine

---

## ▶️ How to Run

```bash

```

2. Open the project folder

```bash
cd Coffee-Vending-Machine
```

3. Run the program

```bash
python vending_machine.py
```
